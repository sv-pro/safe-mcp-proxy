# `demos/assets/`

Assets used by README and docs demos.

| Path | Purpose |
|------|---------|
| `tapes/absent.tape` | VHS source for the absent-tool terminal GIF |
| `tapes/injection.tape` | VHS source for the prompt-injection terminal GIF |
| `generate_gifs.sh` | Regenerates GIFs into `demos/assets/` |

Run:

```bash
bash demos/assets/generate_gifs.sh
```
