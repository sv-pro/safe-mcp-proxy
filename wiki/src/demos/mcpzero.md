# `demos/mcpzero/`

Human-facing MCPZero demo assets.

The MCPZero framework code remains in `mcpzero/`; this directory contains the
canonical demo wrapper and notebook assets.

## Commands

```bash
python -m mcpzero.demo
python -m demos.mcpzero.demo
python -m mcpzero.demo --scenario email_injection
```

## Notebooks

`notebooks/walkthrough.ipynb` is the canonical walkthrough. Older local copies
that were previously scattered under `docs/notebooks/` live in
`notebooks/archive/`.
